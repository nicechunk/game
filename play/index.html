Mining Session

Set mining session SOL

Mining uses integer world coordinates and chunk PDA deltas. This amount is a client-side top-up target for a separate plugin-wallet session key. It is not escrow, a transaction spending cap, or an authorization limit. Local Game Wallet mode uses the Game Wallet balance directly.

  • Rendering never decides resource ownership or mining legality.
  • Only integer coordinate proof data is prepared for chain verification.
  • Pending deltas can be confirmed or rolled back without changing base world data.
